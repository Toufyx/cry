
# cry CLI

Simple CLI to manage encrypted files

## Usage

``` sh
# read an ecrypted file
$ cry read <FILE>

# encrypt a file:
$ cry encrypt <FILE>

# decrypt a file
$ cry decrypt <FILE>
```
